Timestamp Microservice

Documentation

Welcome to the timestamp service. This documentation should supply you with all the information you need to start making requests. It is a really simple api service built to complete one of the projects for the freeCodeCamp Back End Development and APIs certification.

Base Url
https://freecodecamptimestampservice.herokuapp.com
Date Endpoint

An endpoint that will convert a valid date to a json object containing a unix key with a value that is the number of milliseconds equivalent to the passed date and a utc key with a value that is the utc string equivlanet to the passed date.

/api/
Example Request:
https://freecodecamptimestampservice.herokuapp.com/api/2015-12-25
Example Response:
{
 "unix": 1451001600000,
 "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}
